BD vs DBMS

¿Qué es entonces una base de datos - DB?

Base de datos, también llamada base de datos electrónica, cualquier conjunto de datos o información, que está especialmente organizado para la búsqueda y recuperación rápida por un ordenador. Las bases de datos están estructurados para facilitar el almacenamiento, recuperación, modificación y eliminación de datos en conjunción con diversas operaciones de procesamiento de datos.

¿En qué casos es conveniente usar bases de datos?

Es conveniente usar una base de datos cuando un sistema de información requiera de las capacidades de referencia. Una base de datos en el sentido estricto proporciona capacidades de referencias cruzadas. El uso de palabras clave y varios comandos de clasificación, los usuarios pueden buscar rápidamente, reorganizar, grupo y seleccione los campos en muchos registros para recuperar o crear informes sobre determinados agregados de datos.

¿Qué es un sistema de gestión de base de datos - DBMS?

Sistema de gestión de base de datos (DBMS), Sistema para la búsqueda rápida y la recuperación de información de una base de datos.

Enlista y explica las funciones/responsabilidades que tienen los DBMS.

El DBMS determina cómo se almacenan y recuperan datos. Debe abordar problemas tales como la seguridad, la precisión, la consistencia entre los diferentes registros, el tiempo de respuesta, y los requisitos de memoria.

Ejemplifica y justifica en que proyecto de los que has realizado hubiera sido conveniente utilizar una base de datos.

En el proyecto de diseño interactivo, desarrollamos una página web con una lista de restaurantes vegetarianos, sin embargo, no estaba relacionado con nada más. Nos hubiera convenido hacer una base de datos con los restaurantes disponibles dentro de la ciudad de Querétaro para no tener que volver a escribirlos en cada página, sino solo acceder a la base de datos.
