Preguntas

Como ingeniero de software ¿cuál es tu recomendación sobre el uso de !important en un CSS? Las declaraciones !important permiten al autor de un CSS darle más peso a un valor de lo que normalmente tiene. Es un peligroso usar esta declaración. Se debe de reservar su uso para necesidades especiales. Es recomendable evitar su uso cuando sea posible. Más información: https://www.smashingmagazine.com/2010/11/the-important-css-declaration-how-and-when-to-use-it/

Si se pone una imagen de fondo en una página HTML, ¿por qué debe escogerse con cuidado? Porque se va a cambiar de tamaño dependiendo del dispositivo. Si es una imagen pequeña, quizá se vea muy borrosa en una pantalla muy grande.

Como ingeniero de software, ¿cuál es tu recomendación al elegir las unidades de un propiedad de estilo entre %, px y pt? Yo recomendaria usar el porcentaje porque te permite escalar el tamaño de las cosas de manera proporcional, sin depender del tamaño de la pantalla del usuario.

¿Por qué el uso de una versión minimizada del CSS mejora el rendimiento del sitio? Las versiones minimizadas funcionan al analizar y reescribir código para reducir el peso del archivo. Algunos beneficios incluyen:

  1. El contenido carga más rápido
  2. Ocupa menos recursos de internet
  3. Mejora el rendimiento
